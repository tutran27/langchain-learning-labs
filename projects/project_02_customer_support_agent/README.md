# Project 02 · Customer Support Agent

> 💬 Mini project hướng tới agent hỗ trợ khách hàng.

## 🎯 Mục tiêu

- Nhận diện intent của người dùng.
- Sử dụng tool hoặc knowledge base để hỗ trợ trả lời.
- Theo dõi ngữ cảnh nhiều lượt hội thoại.
- Hướng tới quy trình escalation hoặc handoff khi cần.

## 📌 Trạng thái

Project hiện mới ở mức định hướng, chưa có code triển khai cụ thể trong thư mục này.
