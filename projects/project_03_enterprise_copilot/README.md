# Project 03 · Enterprise Copilot

> 🏢 Mini project hướng tới copilot nội bộ cho doanh nghiệp.

## 🎯 Mục tiêu

- Kết nối nhiều nguồn dữ liệu nội bộ.
- Hỗ trợ truy vấn tài liệu, quy trình và kiến thức vận hành.
- Dùng agent workflow để điều phối truy hồi, tool use và tổng hợp câu trả lời.
- Tăng khả năng kiểm soát quyền truy cập và audit log.

## 📌 Trạng thái

Project hiện mới ở mức định hướng, chưa có code triển khai cụ thể trong thư mục này.
